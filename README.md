## 🌍 Vision

**OncoSphere** is an AI-powered Cancer Care & Research Management Platform designed to bring patients, doctors, researchers, and healthcare institutions together on one intelligent digital ecosystem.

Our goal is to improve cancer diagnosis, patient management, medical research, collaboration, and healthcare decision-making through modern technology and Artificial Intelligence.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/OncoSphere.git
```

```
cd OncoSphere
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Backend

```bash
cd backend

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```
